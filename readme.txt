**STEPS**

1. Open config.py file as a py file or .txt and setup the configurations as menitioned in comments, But make sure to keep the file saved as .py only
2. Make sure theres sufficent amount for the total transfer ,if not the transaction will fail.
3. save recieving addresses in the address.csv, Do not rename the file.
4. main.py is the source-code file while main.exe is the executable, If any changes were done to main.py use pyinstaller to make a new executable.
5. Openup pycharm, setup a venv file by pip-installing the mentioned modules in main.py
